# DevView Internal Legacy Archive

This directory contains historical migration material retained for audit,
compatibility, and source-history review. It is not part of the public DevView
documentation surface.

Public DevView documentation lives in the top-level README, `docs/index.md`,
`docs/cli-reference.md`, `docs/install.md`, `docs/troubleshooting.md`, and the
current concept entrypoints. Do not link this archive as product guidance.

Archive contents may still use retired terminology, old workflow names, and
legacy storage examples. Treat them as migration inputs until a reviewed cleanup
slice rewrites or removes them.
